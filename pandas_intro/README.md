# Atividade 1 - Exploracao inicial com pandas

## Objetivo

Este notebook apresenta uma analise exploratoria inicial de um dataset de emendas, com enfase nos comandos basicos do `pandas` para leitura, inspecao e entendimento da estrutura dos dados.

## O que foi desenvolvido

- carregamento do dataset a partir de uma fonte externa;
- visualizacao das primeiras 10 linhas e das ultimas 5 linhas;
- identificacao da quantidade de linhas e colunas;
- listagem e interpretacao das colunas da base;
- verificacao de valores nulos;
- analise dos tipos de dados;
- ordenacao por ano para identificar a atualizacao mais recente do conjunto;
- uso de `describe()` para resumo estatistico inicial;
- verificacao de registros duplicados;
- registro de interpretacoes curtas ao longo da atividade.

## Resultado da atividade

O notebook documenta a etapa inicial de reconhecimento da base de dados e consolida praticas fundamentais de analise exploratoria. O resultado serve como base para atividades posteriores de limpeza e transformacao.
